Privacy Policy

Last updated: June 2026

Summary: Spiritual Content Bot does not collect, store, or sell any personal data from end users or viewers. API credentials are stored securely as GitHub Secrets and are never exposed publicly.

1. Overview

This Privacy Policy describes how Spiritual Content Bot ("we", "the Service") handles information when using our automated content pipeline. We are committed to protecting privacy and minimizing data collection.

2. Information We Do NOT Collect

We do not collect any of the following:

  • Personal information from TikTok or YouTube viewers
  • User account credentials beyond what is required for API authentication
  • Browsing history, location data, or device information
  • Any data from individuals who view the posted content

3. Information Used by the Service

The Service uses the following solely for operational purposes:

  • API access tokens — stored as encrypted GitHub Secrets, used only to post content and retrieve basic performance metrics (view counts, likes) for the operated account
  • Performance data — aggregated, non-personal video statistics (views, likes, shares) stored locally in performance_data.json to improve future content generation

4. Third-Party APIs

The Service interacts with the following third-party APIs. Their privacy practices are governed by their own policies:

  • TikTok Privacy Policy
  • Google / YouTube Privacy Policy
  • ElevenLabs Privacy Policy
  • Anthropic Privacy Policy

5. Data Storage & Security

All API credentials are stored as encrypted GitHub Actions Secrets and are never written to logs or exposed in the codebase. Performance data files contain only non-personal, aggregated metrics.

6. Data Retention

Performance data (video metrics) is retained for 90 days via GitHub Actions artifacts, after which it is automatically deleted. No other data is retained.

7. Children's Privacy

The Service is not directed at children under the age of 13. We do not knowingly collect any information from children.

8. Changes to This Policy

We may update this Privacy Policy from time to time. The "Last updated" date at the top of this page reflects the most recent revision.

9. Contact

For privacy-related inquiries, please open an issue on our GitHub repository.
